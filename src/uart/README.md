# UART

ESP32 UART communication implementations demonstrating command-line interfaces and serial communication patterns.

## Projects

### Arduino (`arduino/`)
| Project | Description |
|---------|-------------|
| `uart_basic/` | Basic UART serial communication |
| `uart_advanced/` | Advanced UART with extended features |
| `uart_command_parser/` | Command-line parsing over UART |

### ESP-IDF (`esp-idf/`)
| Project | Description |
|---------|-------------|
| `uart_basic/` | Basic UART using ESP-IDF drivers |
| `uart_advanced/` | Advanced UART with extended features |

## Tools

- **Board:** ESP32-WROOM-32E
- **IDE:** VSCode + PlatformIO
- **Compatibility:** Arduino IDE, native `idf.py` toolchain
