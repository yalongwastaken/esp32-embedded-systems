# ADC

ESP32 ADC implementations demonstrating analog sensor monitoring, signal processing, and PWM-based LED control.

## Projects

### Arduino (`arduino/`)
| Project | Description |
|---------|-------------|
| `adc_basic/` | Basic analog-to-digital conversion |
| `adc_advanced/` | Advanced ADC with signal processing |

### ESP-IDF (`esp32/`)
| Project | Description |
|---------|-------------|
| `adc_basic/` | Basic ADC using ESP-IDF drivers |
| `adc_advanced/` | Advanced ADC with signal processing |

## Tools

- **Board:** ESP32-WROOM-32E
- **IDE:** VSCode + PlatformIO
- **Compatibility:** Arduino IDE, native `idf.py` toolchain
